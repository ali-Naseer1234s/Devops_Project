# Screenshot Checklist for Report

Take these screenshots:

1. GitHub repository showing project files.
2. Node.js app running locally at localhost:3000.
3. Docker image build command success.
4. Docker container running locally.
5. Amazon ECR repository with pushed image.
6. EC2 instance running.
7. SSH connection to EC2.
8. `kubectl get nodes` showing Minikube node.
9. `kubectl get pods` showing running pod.
10. `kubectl get services` showing NodePort service.
11. Browser app running at EC2_PUBLIC_IP:30080.
12. GitHub Actions workflow successful.
13. CI/CD update after pushing code change.
14. Scaling demo using `kubectl scale deployment nodejs-devops-app --replicas=2`.
15. Health endpoint at EC2_PUBLIC_IP:30080/health.
